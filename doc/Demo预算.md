# 绿境智慧 AI 园丁 — Demo 项目预算

> **范围定义**：本预算仅覆盖**一次可演示的概念验证 (Demo / PoC)**，不是商业化 MVP，更不是量产计划。
> - **目标交付**：1 个可运行的微信小程序/H5 Demo，覆盖 4 个核心功能：植物识别、土壤拍照判断、个性化养护计划、智能提醒。
> - **周期**：约 10–12 周（4 个里程碑阶段）。
> - **团队**：1 名全栈 + 1 名算法/产品兼职（不计入本预算的现金支出，按"零工资演示版"口径列示，仅核算外购成本）。
> - **口径**：人民币元；面向路演、竞赛、内部立项答辩的最小代价口径。
> - **关联文档**：完整商业化预算见 [项目预算.md](项目预算.md)；功能定义见 [项目计划书 - 详细.md](项目计划书%20-%20详细.md)。

## 5.1 各阶段预算汇总

| 项目阶段 | 预计支出 (元) | 支出项说明 (与里程碑任务对应) |
|---|---:|---|
| **阶段一：需求与方案设计**（第 1–2 周） | **2,200** | 竞品调研、Demo 范围裁剪、原型设计 (Figma 订阅)、技术选型与 API 试用账号开通、需求评审会议杂费 |
| **阶段二：模型部署与开发**（第 3–7 周） | **12,800** | 微信小程序前端骨架开发、后端 (FastAPI) 部署到云服务器、接入植物识别 API、接入大模型 API 生成养护计划、土壤湿度 5 级分类小模型微调（云 GPU 按时计费）、测试数据采集 |
| **阶段三：调优、集成与推广**（第 8–10 周） | **7,500** | 联调与压测、Prompt 与提醒规则调优、识别准确率对比评测、灰度测试用户招募（30 人）、小红书/朋友圈种子内容投放（轻量级） |
| **阶段四：演示与总结**（第 11–12 周） | **2,500** | 路演 PPT/视频制作、Demo 现场所需测试植物与道具、答辩印刷物料、差旅与会务、项目总结报告 |
| **合计支出预算** | **25,000** | 含约 8% 不可预见费缓冲 |

## 5.3 关键弹性与裁剪策略

| 预算压力 | 优先压缩项 | 可节省 (元) | 后果 |
|---|---|---:|---|
| 总预算降至 1.5 万 | 砍掉小红书投放 + 灰度激励 + 二手手机改借用 | ~6,000 | 失去外部用户验证数据，仅团队内测 |
| 总预算降至 1 万 | 再砍掉土壤样本采集（改用公开数据集） + 病害条目整理 | ~4,000 | 土壤识别准确率下降 10–15%，可能不达演示效果 |
| 总预算升至 4 万 | 增加 2 周联调周期 + 增配 1 名设计师外包 | +15,000 | Demo 完成度显著提升，可直接对接天使路演 |

## 5.4 与商业化预算的衔接

- 本 Demo 通过后，对接 [项目预算.md §P1 MVP 验证期](项目预算.md)（25 万元口径），核心增量是**全职团队工资**与**正式知识库建设**。
- 本 Demo 阶段沉淀的资产：土壤湿度标注数据集、Top 50 品种知识条目、植物识别准确率评测基线，**可直接复用进 P1**，避免重复投入。
