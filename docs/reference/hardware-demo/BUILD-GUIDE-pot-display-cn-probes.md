# 盆上环境监测仪（`hardware-demo-pot-display-cn-probes`）落地指南

本文档面向：**要把概念图做成可演示硬件、并具备走向小批量量产路径**的团队。步骤按依赖顺序排列；每步给出**交付物**与**常见坑**，便于排期与验收。

> **与当前软件仓库的关系**：v1 小程序 + 后端已具备「云端天气、拍照估土、计划引擎」等能力。本硬件是**新增本地数据源**（BLE 上报温湿度、光照、盆土相关量等），需在后续独立项目中定义 **设备协议、安全配对、数据入库**；不在本文展开具体 API 设计。

若你需要 **几天内先用面包板走通传感器与屏**（不追求一体机），请优先阅读 **[FAST-DEMO-modular-breadboard.md](./FAST-DEMO-modular-breadboard.md)**；本文侧重 **量产与认证路径**。

## 阶段 0：目标冻结与可行性（约 1～2 周）

### Step 0.1 写一页 PRD（产品需求单）

- **必须写清**：  
  - 显示项：温度、湿度、光照、土壤 pH；是否还要 **盆土含水档位 / EC**（二选一可减复杂度）。  
  - 使用场景：室内盆栽、盆径范围（如 φ12～28 cm）、基质类型（通用营养土 / 颗粒土）。  
  - 续航目标：如「充电一次 ≥ 14 天 @ 每 10 分钟刷新屏显」或「常亮降亮度 + 按键唤醒」。  
  - 连接方式：首版仅 **BLE 5.x** 直连手机，还是 **BLE + Wi-Fi**（后者显著增加认证与功耗设计量）。  
- **交付物**：`PRD-v0.x.pdf` 或仓库内 `docs/.../PRD.md`。

### Step 0.2 技术可行性评审（TDR）

- **pH**：消费级插土 pH 为**难点**（校准、温度补偿、参比电极寿命、肥料/根系干扰）。结论只能是二选一：  
  - **A**：对外宣称「趋势/档位」（3～5 档），接受实验室误差；  
  - **B**：首版 **不做真 pH**，屏上 pH 来自 **算法/用户校准** 或外接笔式电极（会推翻一体造型）。  
- **触脚自稳**：软土、小盆会倒——需否 **可选辅助环/限深卡环** 作为 B 方案写进风险。  
- **交付物**：`TDR-结论.md`（含 Go / No-Go）。

### Step 0.3 预算与里程碑

- 打样预算（结构手板、PCBA 工程批、外壳喷油样品、测试设备如温湿度箱、缓冲液）。  
- **交付物**：甘特图（ID → EVT → DVT → PVT → MP）。

## 阶段 1：工业设计与结构（约 3～6 周，可与电控选型并行）

### Step 1.1 CMF 与外观定稿

- 确定：**金属中框材质**（铝挤 + CNC / 锌合金压铸）、**玻璃**（2.5D 化学强化）、**塑胶内壳**（PC/ABS）、**触脚**（不锈钢 304/316L，表面钝化）。  
- **交付物**：Keyshot / Blender 渲染图 + CMF 表（Pantone / 喷砂目数 / 阳极色号）。

### Step 1.2 结构_layout：触脚 + 止位 + 密封

- **触脚数量**：建议 **3 根**（几何稳定）或 **4 根**（其中 1～2 根为纯结构脚、不布传感器以减少串扰）。  
- **止位**：盆口接触面做 **限深凸台**，防止用户插入过深顶裂盆底或折断探针。  
- **防水**：盆面以上 IPX4～IPX5（泼溅）为合理目标；触脚根部 **硅胶密封圈 + 灌胶** 分区。  
- **交付物**：3D 装配图（STEP）、爆炸图、关键尺寸公差表。

### Step 1.3 手板（3D 打印 + 机加工触脚）

- 验证：**重心 + 三脚** 在目标盆径/基质下的稳定性；屏幕视角是否可读。  
- **交付物**：手板评审纪要（通过 / 改版条目）。

## 阶段 2：硬件架构与原理图（约 4～8 周）

### Step 2.1 系统框图

- 典型划分：**主 MCU** | **显示 + 背光** | **传感器前端** | **电源路径（充电、锂电保护、DCDC）** | **BLE 射频** | **调试口（UART/SWD，量产可封胶）**。  
- **交付物**：框图一页。

### Step 2.2 主控与射频

- 选型方向：**Nordic nRF52/nRF53**、**Silicon Labs EFR32**、或 **ESP32-C3**（若上 Wi-Fi）。首版纯 BLE 建议 **nRF52 系**生态成熟。  
- **天线**：板载陶瓷天线或 FPC，需预留 **匹配网络** 与 **认证测试点**。  
- **交付物**：MCU 模块数据手册归档、射频走线检查清单。

### Step 2.3 传感器清单（与 PRD 对齐）

| 量 | 常见实现思路 | 备注 |
|----|----------------|------|
| 空气温湿度 | 单芯片数字传感器（I²C） | 远离发热元件布局 |
| 光照 | 数字 ALS / 多光谱（成本陡增） | 先做 lux 级；叶面遮挡为产品话术点 |
| 盆土含水 | FDR / 阻抗式（与触脚绝缘布局强相关） | 与 pH 电极需 **空间/时间复用** 或分脚 |
| 盆土温度 | NTC 或数字探头 | 可放在含水探针附近 |
| 土壤 pH | 玻璃膜 pH + Ag/AgCl 参比 | **需模拟前端（低输入偏置运放 + ADC）**、湿存与寿命 |

- **交付物**：BOM v0.1（含备选料）、传感器误差预算表。

### Step 2.4 显示

- **OLED**：观感高档、夜间可读；注意 **烧屏**（UI 反色轮播、像素位移）。  
- **TFT**：彩图 UI 灵活；功耗与户外可读性要权衡。  
- **墨水屏**：省电但刷新慢、夜读差；适合第二 SKU。  
- **交付物**：屏厂规格书、FPC 弯折与 ESD 防护方案。

### Step 2.5 电源与充电

- 单节 **锂聚合物**（容量按体积）；**锂电保护 IC**、**充电 IC**（USB-C 仅充电或加 USB-PD 协商视成本）。  
- 低功耗策略：BLE 广播间隔、主休眠、屏局部刷新。  
- **交付物**：功耗估算表（mA·h / 日）。

### Step 2.6 原理图 + PCB

- 工具：KiCad / Altium。  
- **布局要点**：模拟前端远离数字与天线；pH 走线 **尽量短、屏蔽、等电位环**；触脚连接器用 **防水母座** 或一体出线。  
- **交付物**：Gerber、BOM、装配图、SMT 坐标文件。

### Step 2.7 工程样机 PCBA（5～20 片）

- 厂内：**上电、烧录、射频无委预扫、各传感器冒烟读数**。  
- **交付物**：《工程样机测试记录》。

## 阶段 3：固件（约 6～10 周，可与硬件并行后半段）

### Step 3.1 板级支持包（BSP）

- GPIO、I²C/SPI、ADC 通道校准（增益/偏移）、充电状态、按键/唤醒。  
- **交付物**：BSP 仓库 + README。

### Step 3.2 传感器驱动与滤波

- 温湿度、光照：厂商参考驱动 + **滑动平均 / 异常值剔除**。  
- 含水：标定 **介质曲线**（至少 2～3 种基质标定块）。  
- **交付物**：标定日志格式（CSV）。

### Step 3.3 pH 测量与校准流程（若 PRD 含真 pH）

- 缓冲液 **pH 4 / 7 / 10** 两点或三点校准；温度补偿模型（简化为查表或线性也可先跑通）。  
- UI：**「进入校准」** 中文引导 + 倒计时浸泡时间。  
- **交付物**：校准 SOP（图文）、工厂校准治具说明。

### Step 3.4 显示 UI（中文）

- 字体：授权商用黑体/宋体子集，减 Flash。  
- 信息架构：主屏四行 + 二级菜单（蓝牙配对、校准、关于）。  
- **交付物**：UI 线框 + 字符串表（中英可预留）。

### Step 3.5 BLE GATT 服务设计

- 定义 **Service/Characteristic UUID**、payload（温湿度、pH×100、电量、错误码）。  
- 安全：配对模式（Just Works / Passkey）、**绑定信息存储**、防重放（可选计数器）。  
- **交付物**：《BLE 协议 v0.1》Markdown。

### Step 3.6 OTA（强烈建议首版预留）

- 双区备份或压缩包校验；失败回滚策略。  
- **交付物**：OTA 流程说明。

## 阶段 4：小程序 / 后端对接（与固件并行）

### Step 4.1 角色划分

- **硬件 → 手机**：BLE 广播 / 连接读数。  
- **手机 → 云端**：小程序 `wx.request` 把聚合数据上报现有后端（需新路由或设备表）。  

### Step 4.2 最小闭环 Demo

- 手机读 BLE → 屏显与小程序 **同屏对比** 同一组数。  
- **交付物**：录屏 Demo + 联调 checklist。

## 阶段 5：可靠性、环境与合规（贯穿 DVT）

### Step 5.1 机械与环境试验

- 跌落（包装态 / 裸机）、插拔寿命、高温高湿存储、低温启动、盐雾（触脚）。  
- **交付物**：试验报告模板 + 首轮数据。

### Step 5.2 电气安全与 EMC（中国区常见路径）

- 带锂电池：参考 **GB 31241** 等现行强制性要求路径（以认证机构清单为准）。  
- **SRRC 无线电发射设备型号核准**（含 BLE 多数情况需办）。  
- **交付物**：认证项目清单、预算、时间表。

### Step 5.3 量产工艺

- **SMT 炉温曲线**、**首件检验**、**气密/淋水抽检**、MES 条码规则。  
- **交付物**：QC 卡、维修手册（触脚更换模块方案）。

---

## 阶段 6：从试产到量产（PVT → MP）

### Step 6.1 封样（Golden Sample）

- 签样：结构、喷油、丝印、屏色温。  
- **交付物**：签样板 2 套 + 签字页。

### Step 6.2 产测治具

- **治具**：pH 校准工位、RF 耦合测试、电流吸收测试。  
- **交付物**：治具 BOM + 软件脚本。

### Step 6.3 MP 爬坡

- 小批量 200～1000 台 → 收集客诉 → 快速 ECO。

## 附录 B：首版「砍需求」清单（若时间与预算紧）

1. 首版砍掉 **真 pH**，仅显示 **酸碱趋势档位** 或由 App 估算。  
2. 首版 **不做 Wi-Fi**，仅 BLE。  
3. 触脚加 **限深卡环** 解决小盆不稳。  
4. 屏幕用 **小 OLED 单色**，把高档感交给 **金属中框 + 玻璃**。
